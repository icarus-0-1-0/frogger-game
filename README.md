# 🐸 Frogger Game

A classic Frogger-style game built with HTML5 Canvas and JavaScript.

## Play Online

Visit: https://pearlweb.com/frogger/

## Features

- 🎮 Classic Frogger gameplay
- 🐢 Moving turtles (that submerge!)
- 🪵 Floating logs
- 🚗 Traffic to avoid
- 📈 Progressive difficulty (speed increases each level)
- 🏆 Score tracking
- ❤️ 3 lives per game

## Controls

- **Arrow Keys** or **WASD** to move
- Reach the green safe zones at the top to complete each level
- Avoid falling in the water or getting hit by cars!

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (no frameworks)
- CSS3 for styling
